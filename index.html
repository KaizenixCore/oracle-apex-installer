üîß Troubleshooting

Error 500 / 574 / 571 / 404?

Run the fix script:

bash ~/oracle-apex-complete/scripts/fix.sh

Services not starting after reboot?

Check systemd services (Linux only):

sudo systemctl status oracle-apex-db sudo systemctl status oracle-apex-ords

GUI not opening?

Install GUI tool (YAD or Zenity):

# Ubuntu/Debian sudo apt-get install yad zenity # openSUSE sudo zypper install yad zenity # Fedora sudo dnf install yad zenity # macOS brew install dialog
